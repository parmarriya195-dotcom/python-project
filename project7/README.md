Project Title:
Multi-Utility Toolkit

Description:
This project is a menu-driven multi-utility toolkit developed using Python.
It demonstrates modular programming, packages, and file handling concepts.

Features:
- Date and Time Operations
- Mathematical Calculations
- Random Data Generation
- UUID Generation
- File Operations (create, write, read, append, delete)
- Module Exploration using dir()

Project Structure:
- main.py : Main controller file
- modules/ : Package containing all utility modules
- __init__.py : Used to make the modules folder a package

How to Run:
1. Open terminal in the project folder
2. Run the command:
   python project7/main.py

Example Use Case:
This toolkit can be used by students or small businesses to generate secure
passwords, assign unique identifiers, manage files, and explore Python modules
efficiently.

Conclusion:
The project demonstrates the use of Python packages, modular programming,
exception handling, and real-world utility applications.
